## ūü§Ē The Problem We're Solving

India has **146 million farmers**. Most of them sell their crops through a chain of 4‚Äď5 middlemen before it reaches the buyer ‚ÄĒ losing **30‚Äď40% of their earnings** at every step.

The trader doesn't know if the farmer is trustworthy. The farmer doesn't know if payment will come. There's **no record, no proof, no recourse.**

**AgroChain fixes this.**

---

## ūüí° What is AgroChain?

AgroChain is a **blockchain-powered agricultural trading platform** that connects farmers directly to traders ‚ÄĒ with every deal, delivery, and payment permanently recorded on the **Shardeum blockchain**.

No middlemen. No broken promises. No missing payments.

> *"The blockchain doesn't forget. Neither do we."*

## ‚ö° Getting Started

### Prerequisites
- Node.js v18+
- MongoDB running locally
- Get free SHM from [Shardeum Faucet](https://faucet-mezame.shardeum.org)

### 1. Clone & Install
```bash
git clone https://github.com/krix2112/AgroChain.git
cd AgroChain
npm install
```

### 2. Environment Setup
```bash
# backend/.env
MONGODB_URI=mongodb://localhost:27017/agrochain
JWT_SECRET=your_jwt_secret
ENCRYPTION_KEY=32characterlongencryptionkeyhere
RELAY_PRIVATE_KEY=your_relay_wallet_private_key
RAZORPAY_KEY_ID=rzp_test_xxxx
RAZORPAY_KEY_SECRET=xxxx
RAZORPAY_WEBHOOK_SECRET=xxxx
CONTRACT_ADDRESS=0xb1330f2e931a3b47f495098b9ECdeeD5b0943c5b
```

### 3. Run Everything
```bash
npx turbo dev
```

This starts the backend, web app, and mobile app simultaneously.
